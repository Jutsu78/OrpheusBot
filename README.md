# OrpheusBot

Discord-бот для автоматичного відстеження та сповіщення про майбутні розпродажі в Steam. Бот збирає актуальні дані безпосередньо з офіційної документації Steamworks.

## Можливості

 Веб-скрапінг документації Steamworks за допомогою axios та cheerio.
 щотижневе оновлення бази даних у пам'яті та щоденна перевірка наявності знижок о 20:00.
 Автоматичне сповіщення користувачів (@everyone) у визначеному каналі при старті розпродажу.

## Стек технологій

 Node.js
 Discord.js (v14)
 Axios
 Cheerio
 Node-cron

## Команди

* !list - виводить повний розклад майбутніх розпродажів.
* !recent - показує поточний активний розпродаж та дату його завершення.
* !next - показує наступний запланований розпродаж та дату його початку.
* !orpheus - відсилка до гри в виді GIF
